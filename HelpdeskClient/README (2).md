# מערכת Helpdesk - ניהול טיקטים

## תיאור המערכת
מערכת לניהול פניות שירות עם 3 סוגי משתמשים:
- **Customer** - פותח טיקטים ומוסיף תגובות
- **Agent** - מטפל בטיקטים שהוקצו אליו
- **Admin** - מנהל את כל המערכת

## טכנולוגיות
- React 18
- React Router v6
- Context API + useReducer
- Axios
- CSS Modules

## הרצת הפרויקט

### שרת
